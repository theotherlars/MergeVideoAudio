The exe file can be detected incorrectly as a virus which is better explained here: https://github.com/MScholtes/PS2EXE

This tool uses ffmpeg to merge the video and audio file you get when you render a Level Sequence in UE5 to one .avi file. 
It is dependant on ffmpeg.exe which can be downloaded from:https://ffmpeg.org/ (or it is located in this repo)
When downloaded, place the ffmpeg.exe file in same location as the Merge-AudioVideoFiles.exe/.ps1
